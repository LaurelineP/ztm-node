PM2

  • Quick start: https://www.npmjs.com/package/pm2
  • Installation: https://www.npmjs.com/package/pm2

    • Commands:

  • pm2 start server.js
    launches server observing the (main?) process
    ( daemonized - able to reload while developing )
  • pm2 stop [ server.js | id ]
    will stop the process
  • pm2 delete server.js
    will kill the process(es) 
  • pm2 list
    will list all your processes
  • pm2 log
    will log in realtime
  • pm2 start server.js -i [ n | max ]
    ( `i` for `instance`): measures the amount of worker
    processes that will be created in our server.
  • pm2 start server
    • ### PM2 and clustering features built-in As it comes with the cluster module we do not need anymore to use the cluster module within our code. - no need to fork inside our JS: pm2 will fork the process - remove code block with ".isMaster" flag identification to fork - t*he code will be run as our worker process.* xw
